Classes vs ID's

What are classes and ids?

Too understand classes and id's, we have to understand what selectors are.
Selectors are used to select the content you want to style there are many different selectors but the most commonly used are id's and classes

What are the key difference between id's and classes

Classes are used when you are wanting to style multiple elements throughout a webpage. For example if you wanted to style mulitple images, text, buttons etc. that have the same styling you can target them all using the class selector

IDs are used when you only want to target a single component. IDs should only be used when you wish to have a specific design to a single element on your page.
