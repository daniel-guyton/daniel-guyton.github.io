Loops is self explanitory it is just instructions that are looped until a condition is met. For example dishes you repeat the step of drying them until they are gone.

What is the DOM?

The DOM (Document Objec Model) is a programming interface. It takes html content and translate into an object so that javascript or other functional programming languages can access and modify the code.

Arrays vs Objects - Accessing data

Arrays and objects are quite frequently used in javascipt so understanding how they work and how to access content inside them is important. An array is a ordered collection of elements mainly values or variables that are indentified by an index (index indicates the position of the element in the array)
Creating an array in javascipt is fairly simple it looks something like 

              
              let array = ['data1', 'data2', 'data3']
              array[1] // this will return 'data2' because the index of an array start at 0

Object, The difference object has between arrays is that the data inside the object are known as properties that conisit of a keys and values. The properties can be maniuplated using .dot notation and []. 

            
              let person = {
                  name: "Daniel"
                  age: 100
                }

                person.name // this will return Daniel using dot notation
                person["name"] // this will return Daniel using the [] notation
